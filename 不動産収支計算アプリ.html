<body>
  <h1>不動産ざっくり収支計算</h1>
  
  <label>物件価格（円）<input type="number" id="price" value="60000000"></label>
  <label>年間想定年収（円）<input type="number" id="annualIncome" value="7592400"></label>
  <label>自己資金（円）<input type="number" id="ownFunds" value="5000000"></label>
  <label>借入期間（年）<input type="number" id="years" value="15"></label>
  <label>借入金利（％）<input type="number" id="interestRate" step="0.01" value="3.0"></label>
  <label>空室率（％）<input type="number" id="vacancyRate" step="0.1" value="10"></label>
  <label>諸経費率（％）<input type="number" id="expenseRate" step="0.1" value="15"></label>

  <button onclick="calculate()">ご試算</button>

  <div class="result" id="result"></div>

  <script>
    function calculate() {
      const price = Number(document.getElementById("price").value);
      const annualIncome = Number(document.getElementById("annualIncome").value);
      const ownFunds = Number(document.getElementById("ownFunds").value);
      const years = Number(document.getElementById("years").value);
      const interestRate = Number(document.getElementById("interestRate").value) / 100;
      const vacancyRate = Number(document.getElementById("vacancyRate").value) / 100;
      const expenseRate = Number(document.getElementById("expenseRate").value) / 100;

      const loanAmount = price - ownFunds;
      const months = years * 12;
      const monthlyRate = interestRate / 12;

      // 元利均等返済：月々の返済額
      const monthlyRepayment = loanAmount * monthlyRate / (1 - Math.pow(1 + monthlyRate, -months));
      const annualRepayment = monthlyRepayment * 12;
      const totalRepayment = annualRepayment * years;

      const adjustedIncome = annualIncome * (1 - vacancyRate);
      const expenses = adjustedIncome * expenseRate;
      const annualOutgo = annualRepayment + expenses;
      const netIncome = adjustedIncome - annualOutgo;

      const surfaceYield = (annualIncome / price * 100).toFixed(2);
      const realYield = (adjustedIncome / price * 100).toFixed(2);
      const postLoanYield = (netIncome / price * 100).toFixed(2);
      const roi = (netIncome / ownFunds * 100).toFixed(2);

      document.getElementById("result").innerHTML = `
        <h2>計算結果</h2>
        借入金額：${loanAmount.toLocaleString()} 円<br>
        月々返済額：約 ${monthlyRepayment.toFixed(0).toLocaleString()} 円<br>
        年間返済額：約 ${annualRepayment.toFixed(0).toLocaleString()} 円<br>
        返済総額：約 ${totalRepayment.toFixed(0).toLocaleString()} 円<br>
        <br>
        家賃収入（空室調整後）：${adjustedIncome.toLocaleString()} 円<br>
        諸経費（年間）：${expenses.toLocaleString()} 円<br>
        年間支出：${annualOutgo.toLocaleString()} 円<br>
        年間手取り：${netIncome.toLocaleString()} 円<br>
        <br>
        表面利回り：${surfaceYield} %<br>
        実質利回り：${realYield} %<br>
        返済後利回り：${postLoanYield} %<br>
        投資利回り（ROI）：${roi} %
      `;
    }
  </script>
</body>
</html>

ChatGPT:

Microsoft Edge単体では直接HTMLファイルを編集する機能はありません。ただし、以下のような方法で編集できます：

✅ 方法1：メモ帳やエディタで編集（おすすめ）

  1. Edgeで表示しているHTMLファイルを 右クリック → 「名前を付けて保存」 で保存
    （またはファイルをすでにダウンロードしている場合はそのままでOK）

  2. 保存したファイルを右クリック → 「プログラムから開く」→「メモ帳」や「Visual Studio Code」などを選択

  3. HTML内の <meta name="viewport" ...> タグを編集

  4. 上書き保存

  5. Edgeで再び開いて確認

✅ 方法2：Visual Studio Code を使う（本格的な編集に便利）

  • 無料エディタ Visual Studio Code を使えば、カラー表示や自動補完で快適に編集できます。

❌ 方法3：Edgeの「開発者ツール」で編集（保存できない）

  • F12キーで開く開発者ツールを使ってHTMLを一時的に編集できますが、「保存」はできません。 → ページをリロードすると元に戻ってしまうので、確認用に使うのが目的になります。

編集すべき内容（例）

html
<!-- 変更前 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- 変更後 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">
